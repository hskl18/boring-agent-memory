# Changelog

All notable changes to this project will be documented in this file.

The project uses semantic versioning once public releases begin.

## 0.1.0 - Unreleased

### Added

- SQLite FTS5/BM25 index over trusted local files.
- CLI commands for init, build, query, status, health, inspect, and stdio serving.
- YAML, TOML, and JSON config loading.
- Explicit include and exclude path handling.
- Privacy-oriented default excludes and secret redaction.
- Python `memory_query()` API.
- Canonical source hash inspection.
- JSON-lines stdio query interface.
- Hermes-style, Codex, and Claude Code examples.
- Documentation for integration, CLI, configuration, API, architecture, privacy, and roadmap.
- Unit tests and local repo-readiness check.
